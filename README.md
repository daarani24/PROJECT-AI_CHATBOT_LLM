# 🤖 AI Chatbot using Groq & Streamlit

## 📌 Overview

This project is a conversational AI chatbot built with **Python**, **Streamlit**, and the **Groq API**. It can answer questions, maintain conversation context, and save chat history using a SQLite database.

## ✨ Features

* 💬 AI-powered conversations
* 🧠 Conversation memory
* 💾 Chat history stored in SQLite
* ⚙️ Adjustable AI settings (Model, Temperature, Max Tokens)
* 🗑️ Clear chat option
* 🔒 Secure API key management using `.env`

## 🛠️ Tech Stack

* Python
* Streamlit
* Groq API (Llama 3.3)
* SQLite
* python-dotenv
